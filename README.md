# Job Portal with Transparent Rejection System

This project is a Django-based job recruitment platform that integrates an ATS system to evaluate resumes and provide transparent feedback to rejected candidates.

## Features

- Resume ATS score calculation
- Missing skill detection
- Transparent rejection explanation
- Employer dashboard
- Job posting system
- Application tracking
- Reconsideration request system

## Technologies Used

- Python
- Django
- PostgreSQL
- Bootstrap
- Chart.js

## How to Run

1. Clone the repository

git clone https://github.com/kalaiselvan-m/job-portal.git

2. Install dependencies

pip install -r requirements.txt

3. Run migrations

python manage.py migrate

4. Run the server

python manage.py runserver
